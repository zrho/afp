Applied Functional Programming

AFP was a class on web programming in Haskell with Yesod at the University of Bonn in winter semester 2013/2014, supervised by Janis Voigtländer. 6 students participated over the course of the semester. The class was two-split in a seminar part with theory talks and three practical browser game projects.

Talks

  • Lazy search, Search/constraint handling combinators (Lukas Heidemann, slides)
  • Test-driven development in Haskell (Lennart Buhl, slides and examples)
  • DSLs in Haskell (Fabian Zaiser, slides)
  • Datatype-Generic Programming (Christian Baumhof)
  • Template Haskell, Quasi-Quoting (Fabian Thorand, slides and examples)
  • GUIs/Functional Reactive Programming (Meike Grewing, slides and handout)
